A journey into PyScript features and challenges behind it.

Why PyScript?

  • Python is almost everywhere.

    We would like to fix that “almost” bit by bringing it literally everywhere!

  • Python is users friendly & easy to learn.

    Kids, students, hobbyists, as well as professionals and/or AI, Python can reach, and be used by, everyone.

PyScript is really easy

Each <script type="m/py"> in the page is a portal for a seamless Python/Web integration.

Any JS related API can be consumed by Python code and vice-versa: modules, packages, libraries ... these all work out of the box!

<script type="mpy">
# MicroPython
from pyscript import window, document

document.body.textContent = f"""
  You are in {window.location.href} !
"""
</script>
